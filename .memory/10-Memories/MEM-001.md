Do not enable vim_mode in Zed configuration. The user's Zed setup is shared with a collaborator who is not familiar with vim. Space-leader keybindings should NOT be used since they only work in vim normal mode. Use standard Ctrl/Alt modifier keybindings instead.

**Why:** The Zed setup serves two audiences — the primary user (who knows vim from Neovim) and a collaborator (who uses standard text editing). Vim mode would confuse the collaborator.

**How to apply:** When configuring Zed keybindings, use Ctrl/Shift/Alt modifier patterns, not vim-style space-leader. The settings.json should have `"vim_mode": false` or omit it entirely. The keymap.json should use Workspace/Editor contexts, not vim_mode contexts.
